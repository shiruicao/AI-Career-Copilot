# 💼 AI Job Copilot — 智能求职决策看板

AI Job Copilot 是一款面向多用户场景、旨在解决高频重复投递痛点的商业级智能求职辅助 SaaS 看板。系统基于 **Streamlit** 前端框架与 **Dify** 深度长链路 AI 工作流搭建，采用无门槛的双端纯文本复制粘贴设计，彻底杜绝多操作系统文件读取的编码冲突，实现 60秒 内多维技能动态对齐与职业红线隐性风险智能探测。

---

## 🚀 核心产品特性

- **双端免上传极速交互**：左侧一键粘贴个人简历画像，右侧无缝贴入全新岗位招聘文本（JD），即用即走，实现物理级零编码冲突。
- **长连接流式传输（Streaming）**：针对 DeepSeek-V4 等长链路、高耗时推理大模型节点进行底层网络穿透优化。采用事件块持续流式报告，彻底攻克 Windows 与局域网环境下传统的 30 秒阻塞断开瓶颈，保证长报告 100% 稳健输出。
- **多维加权打分可视化**：通过高动态雷达图与多维 KPI 卡片，将硬技能匹配（40%）、项目经验（30%）、行业背景（20%）及加分项（10%）进行实时结构化量化对齐。
- **隐性风险红线阻断**：支持用户自主勾选单休、大小周、频繁出差等个人职业红线，AI 智能穿透 JD 文本隐性话术，精准拦截潜在求职陷阱。
- **纯净生产级运行流控制**：全面移除任何硬编码硬契合的假数据与伪默认字典，若后端工作流或物理网络层遭遇中断，前端将百分之百抛出底层真实日志信息，确保决策逻辑切实合理。
- **纯净版决策报告一键下载**：底部折叠长报告支持一键导出为纯净版 `.md` 格式文件，便于用户导入 Notion、Obsidian 或 Word 进行无缝二次编辑，完全杜绝浏览器自带打印的侧边栏杂音。

## 🛠 架构设计与本地部署

系统采用前后端解耦的轻量化穿透架构设计：

### 1. 克隆代码至本地
```bash
git clone <your-repository-url>
cd ai-career-copilot


###2. 补齐本地依赖环境
```Bash
pip install streamlit requests plotly python-dotenv


### 3. 配置本地密钥（仅限协同开发者，普通用户无须配置）
 本系统的核心 AI 推理工作流托管在开发者的专属 Dify 引擎上。
- **如果您是普通使用者**：无须在本地配置任何环境变量，直接使用开发者提供的公网/私有网页端链接，即可一键使用，本地不消耗任何 Token。
- **如果您是协同开发者**：若需在本地调试后端接口代码，请在项目根目录下创建一个名为 `.env` 的文件
（此文件已被 `.gitignore` 自动拦截，绝不泄露至公网），并向项目发起人申请调试密钥：

```env
DIFY_API_KEY="项目发起人提供的专属 Dify 调试密钥"
DIFY_API_URL="[https://api.dify.ai/v1](https://api.dify.ai/v1)"


###4. 运行启动服务
```Bash
streamlit run app.py
